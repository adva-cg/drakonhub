DrakonHub project is closed

Dear DrakonHub user,

I am sad to inform you that DrakonHub will shut down on December the first, 2024. Unfortunately, the DrakonHub project is closed.

All your diagrams stored with DrakonHub will be deleted forever. So please take the necessary actions before December the first, 2024, to save the important information you keep with DrakonHub.

You can download your diagrams to your computer and edit them using the desktop version of DrakonHub.

How to switch to DrakonHub for desktop

It is possible to run your own DrakonHub server. A Linux machine is required for that.

The source code for DrakonHub is here: https://github.com/stepan-mitkin/drakonhub.

The instructions how to install your own DrakonHub server are here.

Проект DrakonHub закрыт

Дорогой пользователь DrakonHub.

С сожалением сообщаем, что сервис DrakonHub будет отключен первого декабря 2024 года. Проект DrakonHub закрыт.

Все ваши диаграммы в DrakonHub будут удалены навсегда. Поэтому, пожалуйста, сохраните ту вашу важную информацию, которую вы храните в DrakonHub.

Вы можете скачать диаграммы из DrakonHub на свой компьютер и редактировать их с помощью DrakonHub для десктопа.

Как перейти на DrakonHub для десктопа

Можно создать собственный сервер DrakonHub. Для этого нужна машина с Линуксом.

Исходный код DrakonHub находится здесь: https://github.com/stepan-mitkin/drakonhub.
