DrakonHub project is closed

Dear DrakonHub user,

I am sad to inform you that DrakonHub will shut down on October the first, 2021. Unfortunately, the DrakonHub project is closed due to the termination of funding.

All your diagrams stored with DrakonHub will be deleted forever. So please take the necessary actions before October the first, 2021, to save the important information you keep with DrakonHub.

How to transfer you diagrams to your private instance of DrakonHub

It is possible to run your own DrakonHub server. A Linux machine is required for that.

The source code for DrakonHub is here: https://github.com/stepan-mitkin/drakonhub.

The instructions how to install your own DrakonHub server are here.

How to transfer your diagrams:

  1. Go to drakonhub.com.
  2. Right-click on a project in the tree view.
  3. Choose "Save to file".
  4. Go to your private instance of DrakonHub.
  5. Create a new project
  6. Right-click on the created project in the tree view.
  7. Choose "Load from file".

Regards

Stepan Mitkin

Проект DrakonHub закрыт

Дорогой пользователь DrakonHub.

С сожалением сообщаю, что сервис DrakonHub будет отключен первого октября 2021 года. Проект DrakonHub закрыт по причине прекращения финансирования.

Все ваши диаграммы в DrakonHub будут удалены навсегда. Поэтому, пожалуйста, сохраните ту вашу важную информацию, которую вы храните в DrakonHub.

Как перенести диаграммы в ваш собственный инстанс DrakonHub

Можно создать собственный сервер DrakonHub. Для этого нужна машина с Линуксом.

Исходный код DrakonHub находится здесь: https://github.com/stepan-mitkin/drakonhub.

Инструкции по установке своего собственного сервера DrakonHub находятся здесь.

Как перенести диаграммы:

  1. Войдите в drakonhub.com.
  2. Щёлкните правой кнопкой мыши на проекте в дереве проектов.
  3. Выберите "Сохранить в файл".
  4. Войдите в свой собственный инстанс DrakonHub.
  5. Создайте новый проект.
  6. Щёлкните правой кнопкой мыши на новом проекте в дереве проектов.
  7. Выберите "Загрузить из файла".
